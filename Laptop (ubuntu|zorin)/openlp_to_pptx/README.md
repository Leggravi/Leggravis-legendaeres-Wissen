## Vorgehen

1. **Songs** aussuchen (ggf. neue Songs **runterladen**)

   1. ggf. **Autoren** bearbeiten (zB. per ***https://www.evangeliums.net/lieder*** suchen) https://songselect.ccli.com/

   2. nicht gefundene Songs ganz einfach **hinzufügen**:

      1. links über dem searchfeld: „add a new song”

      2. dann auf edit all

      3. gegoogelte Lyrics per [Skript](https://grok.com/c/3f740f25-3d5d-4e32-a09f-90c6f132e3ce?rid=be9e3d97-a9f1-47f6-9d5b-9f4202a2f1ec) schön machen

         1. ```
            ich schicke dir lieder und du verwandelst sie infolgendes format!:
            ---[Verse:1]---
            text
             
            ---[Chorus:1]---
            tex
             
            optionen: verse, chorus, bridge, pre-chorus, intro, ending, other
            ```

      4. ggf. Autoren

2. auf **+** drücken (zu „service” hinzufügen)

3. ggf. Strophen **Reinfolge** ändern

4. **Theme** auswählen

5. **speichern**
